# Evidencia - Java Consultorio

## 1. Instalación y configuración
Para ejecutar este proyecto, clona el repositorio e impórtalo en IntelliJ IDEA como un proyecto Maven/Java. Asegúrate de tener instalado Java JDK 17 o superior.

## 2. Uso del programa
Ejecuta la clase `Main.java`. El programa cargará el menú interactivo en la consola para la gestión de médicos, pacientes y citas. Los datos se guardan automáticamente en la carpeta local `db`.
